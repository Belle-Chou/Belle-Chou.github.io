给生活修学分-2017.04.04

Mar 22, 2017 | 成长计划 | Hits

近来的日子过的毫无头绪，现在生活渐渐趋于平静，我需要梳理一下生活内容，重新拾起搁置了很久的kindle ，定下阅读计划保证高质量的输入和输出。同时，我还需要利用平时的时光学习一些生活技能，比如盆栽、做饭、整理房间等等。已经报名的信息系统项目管理师考试还有一个多月，我只有两周的时间来计划平日的生活作息，过了这两周我就要开始全心复习考试了，接下来还会有教师资格证考试，这次一定要过啦~

2017年04月第01周计划

每周阅读一本书，做好书评或读书笔记

2017年4月第1周阅读《IT项目经理成长手记》，这一类的书籍需要做好读书笔记。

掌握一门生活技能，做好笔记

2017年4月第1周自己摸索整理房间，把东西安排布置的仅仅有条并购置生活必需品。

  • 更新一篇生活或者工作感悟

但愿2017年4月第1周可以顺利的辞职并且入职新公司，了解新公司的相关工作日常。

kindle里不能多于三本书

现在kindle 里面只有《IT项目经理成长手记》和《猫鼠游戏》，第3本我希望下载一本经典的文学作品，比如《我与地坛》或者《红楼梦》这类的。在新公司我也想坚持写一份工作手记，只是现在我还不知道该是什么岗位的成长手记。

学习英语，背诵单词

2017年4月第1周我只要做好一个英语学习的方法就可以了，具体操作在下一周开始。
